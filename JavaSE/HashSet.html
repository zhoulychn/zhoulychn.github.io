HashSet确实也是用得比较多的一个集合类，但是他的功能实现十分简单，全部依赖于HashMap，元素全部为HashMap的key，保证不重复即可。如果对HashMap不了解，可以先去看看。

  1. //主要属性如下
  3. private transient HashMap<E,Object> map;
  5. private static final Object PRESENT = new Object();
  7. //默认构造函数，创建一个HashMap对象。
  8. public HashSet() {
  9.     map = new HashMap<>();
  10. }

(1) add

添加元素，调用HashMap的put方法，value传入一个静态obj属性。

  1. public boolean add(E e) {
  2.     return map.put(e, PRESENT)==null;
  3. }

(2) remove

删除元素，调用HashMap的remove方法，并判断value值是否为PRESENT对象。

  1. public boolean remove(Object o) {
  2.     return map.remove(o)==PRESENT;
  3. }

(3) clear

清空所有元素，调用HashMap的clear方法。

  1. public void clear() {
  2.     map.clear();
  3. }

(4) contains

判断是否包含某个元素，调用HashMap的containsKey方法。

  1. public boolean contains(Object o) {
  2.     return map.containsKey(o);
  3. }

(5) size

获取集合的元素个数，调用HashMap的containsKey方法。

  1. public int size() {
  2.     return map.size();
  3. }

(6) isEmpty
