问题描述

给定一个整数数组，找到一个具有最大和的子数组，返回其最大和。(PS:子数组最少包含一个数)。

算法思路

贪心

扫描数组，定义变量，sum：扫描中局部子数组和，result：最大的子数组和。

如果当前元素为正数，加上sum必然会更大，所以将该元素纳入子数组中。

如果当前元素为负数，则需要丢弃，sum暂时为局部最大，需要将保存结果到result，由于result可能保存过之前的局部最大子数组，所以两者取较大。

  1. private int maxSubArray(int[] nums) {
  3.     if (nums == null || nums.length == 0) return 0;
  5.     int result = nums[0], sum = nums[0] < 0 ? nums[0] : 0;
  7.     for (int num : nums) {
  9.         //进行比较，重置result
  10.         if (num < 0){
  11.             result = Math.max(result, sum);
  12.         }
  14.         //判断num是否该纳入
  15.         sum = Math.max(num, sum + num);  
  16.     }
  18.     //result在循环过程中出现负数才可能重置，最后一次统计完sum没出现负数结果就不能及时更新。
  19.     return Math.max(result, sum);
  20. }
