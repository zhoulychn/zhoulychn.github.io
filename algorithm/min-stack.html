问题描述

实现一个带有取最小值min方法的栈，min方法将返回当前栈中的最小值。

你实现的栈将支持push，pop 和 min 操作，所有操作要求都在O(1)时间内完成。

算法思路

构造两个栈A，B。前者保存数据，后者保存最小值的索引。

push：如果时第一次压入栈，这个数就是最小值，压入两个栈。后续元素如果小于当前最小值，则先入数据栈，然后将其索引压入最小值栈

pop：如果出栈的元素时最小值，则把其索引在最小值栈中清除。

代码实现

  1. private int[] data = new int[64];
  3. private int[] minList = new int[64];
  5. private int top = -1;
  7. private int minTop = -1;
  9. public void push(int number) {
  10.     data[++top] = number;
  12.     //判断最小值进栈的两种情况
  13.     if (minTop == -1 || number < data[minList[minTop]]) {
  14.         minList[++minTop] = top;
  15.     }
  16. }
  19. public int pop() {
  21.     //判断是否最小值
  22.     if (top == minList[minTop]) {
  23.         minTop--;
  24.     }
  25.     return data[top--];
  26. }
  28. //直接取最小值
  29. public int min() {
  30.     return data[minList[minTop]];
  31. }
